# SpendNote - 轻量纯净的记账小程序

一款基于微信小程序云开发的记账应用，主打轻量、纯净、数据自主。

## 功能特性

- **极简记账** - 3步完成一笔记账：金额 → 分类 → 完成
- **收支管理** - 支持收入和支出记录，预设常用分类
- **月度统计** - 按月查看收支概览和分类明细
- **数据安全** - 所有数据存储在微信云开发，安全可靠
- **无广告** - 纯净体验，专注记账

## 技术栈

- 前端：微信小程序原生开发
- 后端：微信云开发（云函数 + 云数据库）
- 架构：Serverless

## 快速开始

### 1. 环境准备

- 微信开发者工具
- 微信小程序账号
- 开通云开发

### 2. 项目配置

1. 克隆项目
2. 在 `miniprogram/app.js` 中配置云开发环境 ID
3. 在云开发控制台创建 `bills` 和 `categories` 集合
4. 上传云函数 `billFunctions`

### 3. 运行项目

1. 使用微信开发者工具打开项目
2. 点击「预览」或「真机调试」

详细部署步骤请参考 [部署指南](docs/deploy-guide.md)

## 项目结构

```
SpendNote/
├── cloudfunctions/          # 云函数目录
│   └── billFunctions/       # 账单相关云函数
├── miniprogram/             # 小程序前端目录
│   ├── pages/               # 页面目录
│   │   ├── index/           # 首页（账单列表）
│   │   ├── add/             # 记账页面
│   │   ├── stats/           # 统计页面
│   │   └── mine/            # 我的页面
│   └── images/              # 静态资源
├── docs/                    # 开发文档
└── tools/                   # 工具脚本
```

## 竞品分析

本项目参考了市面上主流记账小程序的特点：

| 竞品 | 特点 | SpendNote 的差异化 |
|------|------|-------------------|
| 随手记 | 功能全面但臃肿 | 轻量简洁 |
| 鲨鱼记账 | 极简记账 | 同样极简，更强统计 |
| 钱迹 | 无广告纯净 | 同样纯净，云端同步 |
| 微信自带记账 | 与支付打通 | 更强功能，自定义分类 |
