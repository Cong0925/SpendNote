## 导入方式

### 方式1：通过云函数导入

```javascript
// 在微信开发者工具控制台执行
const fs = wx.getFileSystemManager()
const data = fs.readFileSync('path/to/test-yearly-summaries.json', 'utf8')
const lines = data.split('\n').filter(line => line.trim())

for (const line of lines) {
  const record = JSON.parse(line)
  await wx.cloud.callFunction({
    name: 'yearlySummary',
    data: {
      action: 'import',
      record
    }
  })
}
```

### 方式2：通过数据库控制台导入

1. 打开微信开发者工具
2. 进入云开发控制台
3. 选择数据库集合 `yearly_summaries`
4. 点击导入按钮
5. 选择 `test-yearly-summaries.json` 文件

## 注意事项

1. 导入前确保数据库集合 `yearly_summaries` 已创建
2. 测试数据使用固定的 `_openid`（oFEshxuHd5E6Ikbhbpkz4WrVGI0c），与当前用户一致
3. 2026年数据为当前年份的部分数据（截至7月）
