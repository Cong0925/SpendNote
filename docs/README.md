## 项目简介

SpendNote 是一个基于微信小程序云开发的记账应用，使用云函数、云数据库和云存储等能力。

## 技术栈

- 微信小程序原生框架
- 微信云开发（CloudBase）
- 云函数（Node.js）
- 云数据库（MongoDB）
