# 账户模块数据库设计

## 集合列表

### 1. accounts（账户表）

存储用户的所有账户信息

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| _id | string | 是 | 自动生成的唯一标识 |
| _openid | string | 是 | 用户openid（自动关联） |
| name | string | 是 | 账户名称（如：微信、招商银行） |
| type | string | 是 | 账户类型：cash/wechat/alipay/qq/savings/credit/huabei/jd/mt/other |
| icon | string | 是 | 图标路径 |
| balance | number | 是 | 余额（资产类）或应还金额（负债类） |
| isDebt | boolean | 是 | 是否为负债类账户 |
| bankName | string | 否 | 银行名称（仅银行卡类账户） |
| bankCardLast4 | string | 否 | 银行卡号后四位（仅银行卡类账户） |
| creditLimit | number | 否 | 信用额度（仅信用卡） |
| billDay | number | 否 | 账单日，每月1-31（仅信用卡） |
| repayDay | number | 否 | 还款日，每月1-31（仅信用卡） |
| remark | string | 否 | 备注 |
| sortOrder | number | 否 | 排序顺序（默认0） |
| createdAt | date | 是 | 创建时间 |
| updatedAt | date | 是 | 更新时间 |

**索引：**
- _openid + type
- _openid + isDebt
- _openid + sortOrder

**账户类型说明：**
- cash: 现金
- wechat: 微信
- alipay: 支付宝
- qq: QQ钱包
- savings: 储蓄卡
- credit: 信用卡
- huabei: 花呗
- jd: 京东白条
- mt: 美团月付
- other: 其他

### 3. loans（借款记录表）

存储用户的借款记录

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| _id | string | 是 | 自动生成的唯一标识 |
| _openid | string | 是 | 用户openid（自动关联） |
| type | string | 是 | 借款类型：lend（借出）/ borrow（借入） |
| amount | number | 是 | 借款总金额 |
| paidAmount | number | 是 | 已收/已还金额（默认0） |
| personName | string | 是 | 借给谁/谁借的 |
| accountId | string | 否 | 关联账户ID |
| loanDate | date | 是 | 借款日期 |
| dueDate | date | 否 | 预计还款日期 |
| remark | string | 否 | 备注 |
| images | array | 否 | 图片列表（最多4张） |
| status | string | 是 | 状态：pending（进行中）/ completed（已完成）/ cancelled（已取消） |
| createdAt | date | 是 | 创建时间 |
| updatedAt | date | 是 | 更新时间 |

**索引：**
- _openid + type
- _openid + status
- _openid + loanDate

## 数据关系

```
用户（_openid）
    ├── accounts（账户列表）
    │       ├── 资产类账户（balance > 0）
    │       └── 负债类账户（balance > 0，表示应还金额）
    └── loans（借款记录）
            ├── 借出记录（type = lend）
            └── 借入记录（type = borrow）
```

## 计算公式

- **总资产** = 所有资产类账户 balance 之和
- **总负债** = 所有负债类账户 balance 之和
- **净资产** = 总资产 - 总负债
- **借出应收** = 所有借出记录 (amount - paidAmount) 之和
- **借入应还** = 所有借入记录 (amount - paidAmount) 之和
