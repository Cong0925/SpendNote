# 转账记录数据库设计

## 集合列表

### transfers（转账记录表）

存储用户的账户间转账记录

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| _id | string | 是 | 自动生成的唯一标识 |
| _openid | string | 是 | 用户openid（自动关联） |
| fromAccountId | string | 是 | 转出账户ID |
| toAccountId | string | 是 | 转入账户ID |
| amount | number | 是 | 转账金额 |
| date | string | 是 | 转账日期（YYYY-MM-DD） |
| remark | string | 否 | 备注 |
| createdAt | date | 是 | 创建时间 |

**索引：**
- _openid + date
- _openid + fromAccountId
- _openid + toAccountId

## 数据关系

```
用户（_openid）
    └── transfers（转账记录）
            ├── fromAccountId → accounts._id（转出账户）
            └── toAccountId → accounts._id（转入账户）
```

## 注意事项

1. **余额更新**：添加转账记录时，自动更新转出和转入账户的余额
2. **删除恢复**：删除转账记录时，自动恢复转出和转入账户的余额
3. **账户验证**：转出和转入账户不能相同
