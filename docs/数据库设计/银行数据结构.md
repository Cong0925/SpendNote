# 银行数据结构设计

## 集合字段

### banks（银行表）

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| _id | string | 是 | 自动生成的唯一标识 |
| name | string | 是 | 银行全称（如：中国工商银行） |
| shortName | string | 是 | 银行简称（如：工商银行） |
| pinyin | string | 是 | 拼音标识（用于搜索，如：icbc） |
| category | string | 是 | 分类：state-owned/joint-stock/city-commercial/rural-commercial/foreign/other |
| isHot | boolean | 是 | 是否热门（前30家） |
| sortOrder | number | 是 | 排序序号 |
| icon | string | 是 | 图标路径 |
| createdAt | date | 是 | 创建时间 |

## 分类说明

| 分类 | 英文标识 | 说明 |
|------|----------|------|
| 国有银行 | state-owned | 6家 |
| 股份制银行 | joint-stock | 12家 |
| 城市商业银行 | city-commercial | 100+家 |
| 农村商业银行 | rural-commercial | 50+家 |
| 外资银行 | foreign | 20+家 |
| 其他 | other | 10+家 |

## 索引设计

- name（唯一）
- category + sortOrder
- isHot + sortOrder
- pinyin（用于搜索）

## 查询策略

1. **首屏加载**：只查询 isHot=true 的银行（约30家）
2. **分类展开**：查询指定 category 的银行
3. **搜索**：使用 pinyin 或 name 进行模糊查询，限制返回20条
