# 更新日志

本文件记录 SpendNote 项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.0.0] - 2026-06-30

### 新增

#### 核心功能
- 记账功能：支持收入和支出记录
- 分类管理：预设 16 个常用分类（10 支出 + 6 收入）
- 账单列表：按时间展示账单，支持按月筛选
- 统计分析：按月统计收支，分类明细展示
- 数据管理：查看总览、清除数据

#### 页面
- 首页：账单列表，收支概览
- 记账页：金额输入，分类选择，日期选择
- 统计页：收支统计，分类明细
- 我的页：用户信息，数据总览，设置菜单

#### 技术实现
- 云函数：billFunctions（增删改查）
- 云数据库：bills、categories 集合
- UI 设计：渐变主题色，卡片式布局
- 交互优化：加载状态，空状态提示

#### 文档
- 架构说明文档
- 数据库设计文档
- 部署指南
- 图标获取说明
- 更新日志
