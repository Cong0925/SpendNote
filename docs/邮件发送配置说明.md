# 邮件发送配置说明

## 概述

问题反馈功能支持将用户反馈自动发送到指定邮箱。使用 SendGrid 作为邮件发送服务。

## 配置步骤

### 1. 注册 SendGrid 账号

1. 访问 [SendGrid 官网](https://sendgrid.com/)
2. 注册账号（免费额度：100封/天）
3. 验证邮箱

### 2. 创建 API 密钥

1. 登录 SendGrid 控制台
2. 进入 Settings → API Keys
3. 点击 "Create API Key"
4. 选择 "Full Access" 或 "Mail Send" 权限
5. 复制生成的 API 密钥

### 3. 在云开发控制台配置环境变量

1. 打开微信开发者工具
2. 点击「云开发」按钮
3. 进入「设置」→「环境变量」
4. 添加环境变量：
   - 名称：`SENDGRID_API_KEY`
   - 值：您的 SendGrid API 密钥

### 4. 验证域名（可选但推荐）

1. 在 SendGrid 控制台进入 Settings → Sender Authentication
2. 添加您的域名
3. 按照提示配置 DNS 记录
4. 等待验证通过

## 工作原理

### 定时触发器

云函数配置了定时触发器，每天早上9点自动执行：

```json
{
  "triggers": [
    {
      "name": "emailTrigger",
      "type": "timer",
      "config": "0 0 9 * * * *"
    }
  ]
}
```

### 发送流程

1. 查询所有 `emailSent: false` 的反馈（最多100条）
2. 按用户分组
3. 获取用户信息
4. **下载图片并转换为base64**
5. 每封邮件最多30条反馈，超过分批发送
6. 生成 HTML 邮件内容
7. 调用 SendGrid API 发送邮件
8. 标记 `emailSent: true`

### 邮件内容

邮件包含：
- 用户信息（昵称、头像）
- 反馈类型、内容、截图（base64嵌入）
- 反馈编号、提交时间
- 联系方式（如有）

### 图片处理

**问题：** 云存储的图片URL无法被外部访问

**解决方案：** 将图片下载并转换为base64格式，直接嵌入邮件

```javascript
// 1. 从云存储下载图片
const fileRes = await cloud.downloadFile({ fileID: imageUrl })

// 2. 转换为base64
const base64 = fileRes.fileContent.toString('base64')

// 3. 嵌入邮件
<img src="data:image/png;base64,${base64}">
```

**优点：**
- 图片直接显示，无需外部访问
- 邮件客户端兼容性好
- 保护图片隐私

**缺点：**
- 邮件体积增大
- 每封邮件大小限制 30MB

### 分批发送

每封邮件最多包含30条反馈：

```javascript
const maxFeedbacksPerEmail = 30

for (let i = 0; i < feedbacks.length; i += maxFeedbacksPerEmail) {
  const batch = feedbacks.slice(i, i + maxFeedbacksPerEmail)
  await sendEmail(batch)
}
```

**示例：**
- 45条反馈 → 发送2封邮件（30条 + 15条）
- 100条反馈 → 发送4封邮件（30+30+30+10）

## 安全说明

- API 密钥存储在云函数环境变量中，不会暴露给前端
- 邮件发送在云函数中执行，用户无法直接调用
- 只有未发送的反馈会被处理

## 常见问题

### Q: 邮件没有发送？

A: 检查以下几点：
1. 环境变量 `SENDGRID_API_KEY` 是否配置正确
2. SendGrid 账号是否激活
3. 云函数是否部署成功
4. 定时触发器是否正常工作

### Q: 如何手动触发邮件发送？

A: 可以在云开发控制台手动调用云函数：
```javascript
wx.cloud.callFunction({
  name: 'feedbackFunctions',
  data: { action: 'sendPendingEmails' }
})
```

### Q: 如何查看发送记录？

A: 在 SendGrid 控制台的 Activity 页面可以查看所有发送记录。

## 替代方案

如果不想使用 SendGrid，可以考虑：

1. **腾讯企业邮** - 免费，需要配置域名
2. **阿里企业邮** - 免费，需要配置域名
3. **微信订阅消息** - 免费，但只能发送给用户
