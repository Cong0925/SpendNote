关于开销随手记

开销随手记（SpendNote）是一款开源的微信记账小程序。 我们相信，记账不应该是件麻烦事。

通过简洁的界面设计和实用的统计功能，让每个人都能轻松管理自己的财务。 作为开源项目，欢迎社区参与贡献和完善。
